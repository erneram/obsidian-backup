# 🏥 Clinic Management – Brainstorm

Sistema de gestión clínica. Aún en fase de planeación/brainstorm.

## 💡 Ideas y Decisiones

### Frontend
- **PWA** — página que se instala pero funciona como web app.
- **TanStack Query** — para manejo de estado asíncrono y fetching.
- **Vue 3** o **React** con soporte para REST o GraphQL.

### Backend (opciones)
- Go
- .NET con C#
- Node.js
- GraphQL

## 📋 Funcionalidades

- **Calendario de Citas** con estados:
  - Programado → Sala de espera → En consulta → Terminado → Cancelado
- **Notificaciones automáticas** — correo o WhatsApp al programar cita.
- **Recetas** — sistema de seguimiento.
- **Archivos adjuntos** — documentos por paciente.
